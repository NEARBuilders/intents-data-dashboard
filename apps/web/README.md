# Intents Data Dashboard

This is the frontend application for the Intents Data Dashboard, built with [TanStack Start](https://tanstack.com/start).

## Getting Started

To run this application:

```bash
bun install
bun run start
```

## Building For Production

To build this application for production:

```bash
bun run build
```

## Testing

This project uses [Vitest](https://vitest.dev/) for testing. You can run the tests with:

```bash
bun run test
```
